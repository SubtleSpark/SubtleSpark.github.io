# SubtleSpark Homepage

A Vite + React + Tailwind CSS personal homepage, deployed to GitHub Pages.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deployment

Push to `main` to trigger `.github/workflows/deploy.yml`.

This repository is configured as a GitHub user site (`SubtleSpark.github.io`) and uses Vite `base: '/'` for root-domain deployment.
